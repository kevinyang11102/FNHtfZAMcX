# 健身房管理系统 java994

## 项目概述

健身房管理系统是一款基于Java技术栈的全面管理软件，旨在帮助健身房简化管理流程，提高工作效率。

## 技术栈

- 前端：Vue
- 后端：SpringBoot
- 数据库：MySQL
- 开发环境：Eclipse/IntelliJ IDEA/MyEclipse
- JDK版本：JDK 8

## 功能模块

系统包含但不限于以下功能模块：
- 会员管理
- 课程预约
- 教练管理
- 财务管理
- 设备管理

## 系统角色

- 系统管理员
- 前台接待
- 教练
- 会员

## 运行环境

- 数据库：MySQL
- 服务器：任何支持Java Web的服务器
- 客户端：现代Web浏览器

## 部署步骤

1. 准备好MySQL数据库环境
2. 导入SQL脚本
3. 部署后端代码到服务器
4. 部署前端代码到Web服务器
5. 启动后端服务

  - 前端代码目录 (Vue项目)
  - 后端代码目录 (SpringBoot项目)
  - sql脚本目录
